# 서울 방랑자

AI와 여러 API를 활용한 서울 내 여행지 추천 심리테스트 앱

## 주요 기능

- 개인 취향과 상황 기입
- 동행자 취향과 상황 (최대 2인까지 추가 가능)
- 서울데이터 API로 서울 내 지역 추천
- OpenAI로 추천 이유 제시와 해당 지역 내 명소 등 추천
- 관광공사 API로 지역 별 사진 제공

## 사용 방법

1. 앱 접속
2. 사이드바에 API 키 입력
3. 메인 화면 필요 정보 기입
4. AI 분석 결과 확인

## 사용된 API

- [서울 열린데이터 광장](https://data.seoul.go.kr/)
- [한국관광공사 포토코리아](https://www.data.go.kr/data/15101914/openapi.do)
- [OpenAI](https://openai.com/)
